# react-native-simple-yamap

Yandex simple maps for react native

## Installation


```sh
npm install react-native-simple-yamap
```


## Usage


```js
import { SimpleYamapView } from "react-native-simple-yamap";

// ...

<SimpleYamapView color="tomato" />
```
